# ロージー - 犬インフルエンサー向けWebサイト

React + Vite + Tailwind CSS で構築されたポートフォリオサイト

## セットアップ

依存関係のインストール
```
npm install
```

## 開発サーバーの起動

```
npm run dev
```

## ビルド

```
npm run build
```

ビルド後、distフォルダが生成されます

## プレビュー

```
npm run preview
```

## デプロイ

Cloudflare Pagesにデプロイする場合
1. distフォルダの内容をアップロード
2. _redirectsファイルが含まれていることを確認

## 技術スタック

- React 19.2.0
- Vite 7.2.4
- Tailwind CSS
- React Router
- Formspree（フォーム送信）

## ページ構成

- TOP: メインビジュアルとキャッチコピー
- About: ロージーの紹介（写真2枚、テキスト50文字以内）
- Gallery: 写真・動画ギャラリー
- SNS: Instagramリンク
- Works: PR実績
- Contact: お仕事依頼フォーム

## 改善点

- 落ち着いた配色に変更（ポップすぎるデザインを改善）
- 画像の遅延読み込み実装
- フォームバリデーション追加
- アクセシビリティ向上（フォーカス状態、ARIA属性）
- モバイルナビゲーション改善（ハンバーガーメニュー）
- ローディング状態の視覚的フィードバック

## 注意事項

- ContactページのFormspreeフォームIDを設定してください
- 画像は現在プレースホルダーを使用しています。実際の画像に差し替えてください
- InstagramのリンクURLを実際のアカウントに変更してください
