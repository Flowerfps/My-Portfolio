画像ファイルをここに配置してください

## 推奨されるファイル名

- TOPページのメイン画像: `top-main.jpg` または `top-main.png`
- Aboutページの写真1: `about-1.jpg` または `about-1.png`
- Aboutページの写真2: `about-2.jpg` または `about-2.png`
- Galleryページの写真: `gallery-1.jpg`, `gallery-2.jpg`, ... など

## 使い方

コード内では以下のようにパスを指定します：
```jsx
<img src="/images/top-main.jpg" alt="ロージー" />
```

## 注意事項

- 画像ファイル名は英数字とハイフン（-）を使用してください
- 日本語のファイル名は避けてください
- ファイルサイズが大きい場合は、事前に最適化することをお勧めします
